## Personal Site

Live: https://personal-site-jv04.onrender.com/en

Status: Still in development. Features and visuals are evolving.

### Tech stack

- Next.js (App Router) for routing, pages, and SSR
- React + TypeScript for UI and type safety
- Tailwind CSS + CSS variables for theming and design tokens
- next-intl for i18n and locale-based routing
- next-themes for theme handling (currently fixed dark mode)
- Web Workers + OffscreenCanvas for background animations (fireworks)
- Emscripten/WASM assets for the Bunny Run simulation

### Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000
