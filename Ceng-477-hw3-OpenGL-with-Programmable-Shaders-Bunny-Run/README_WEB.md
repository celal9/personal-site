# WebGL (Emscripten) build

This project can run in the browser after a WebGL2 port via Emscripten.

## 1) Install Emscripten
- https://emscripten.org/docs/getting_started/downloads.html

## 2) Build
From this folder:

```
emcc main.cpp -O2 \
  -s USE_GLFW=3 \
  -s FULL_ES3=1 \
  -s MIN_WEBGL_VERSION=2 \
  -s MAX_WEBGL_VERSION=2 \
  -s ALLOW_MEMORY_GROWTH=1 \
  -s ENVIRONMENT=web \
  -s MODULARIZE=0 \
  -s EXPORT_ES6=0 \
  -o bunny-run.js \
  --preload-file sky.jpg \
  --preload-file bunny.obj \
  --preload-file armadillo.obj \
  --preload-file cube.obj \
  --preload-file quad.obj \
  --preload-file vert.glsl \
  --preload-file frag.glsl \
  --preload-file vert2.glsl \
  --preload-file frag2.glsl \
  --preload-file vert3.glsl \
  --preload-file frag3.glsl \
  --preload-file vert4.glsl \
  --preload-file frag4.glsl \
  --preload-file vert5.glsl \
  --preload-file frag5.glsl
```

## 3) Copy output to the website
Place the output files here:

`public/simulations/bunny-run/`

Expected files:
- `bunny-run.js`
- `bunny-run.wasm`
- `bunny-run.data` (if generated by preload)

The project page will load `/simulations/bunny-run/bunny-run.js` automatically.
